# Executive Brief Package: Clinical Roundup → AI-Native CTMS

This folder contains the same executive brief in two publish-ready formats:

1. **Markdown source** (for GitHub/web viewing and version control)
   - `Clinical-Roundup-CTMS-Exec-Brief.md`
2. **Word-compatible document** (HTML saved with `.doc` extension)
   - `Clinical-Roundup-CTMS-Exec-Brief.doc`

## Download and open the Word file

### Option A — from GitHub file view
1. Open `Clinical-Roundup-CTMS-Exec-Brief.doc` in GitHub.
2. Click **Download raw file**.
3. Open the downloaded `.doc` in Microsoft Word.
4. In Word, choose **File → Save As** and save as `.docx` if needed.

### Option B — from a raw URL
1. Open the raw file URL for the `.doc` file.
2. Right-click and choose **Save link as…**.
3. Open in Microsoft Word.
4. Save as `.docx` from Word.

## Notes

- The `.doc` file is intentionally Word-compatible HTML with Microsoft Office namespaces.
- It is formatted for executive review (cover page, static table of contents page, section page breaks, callout boxes, and native tables).
- No external assets are required.
